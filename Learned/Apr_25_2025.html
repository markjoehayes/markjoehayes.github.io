So, what did I learn this week? I tried to focus mostly on review and finish my shell project, but I did manage to get some new concepts in. Specifically randomizing in C. Apparently, in C (all languages?) nothing can be truly random - even with rand() from stdlib.h, the number choices will the same each time you run the program on the same mation. To get around that, I learned to use a seed number, specifically through time() from time.h, which defaults through Unix Epoch Time. As long there is a second between runs, the number will be different and mostlyrandom.
